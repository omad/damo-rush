Dino-Rush

Get your personalized puzzle deck for the rush hour game from a 2 million+ puzzles database !

This site let you generate files to print you own deck of puzzle. The format is especially designed so that you can order real card on specific website so that you'll end up with something like this:

The database used include "all interresting puzzle". You only need to provide the number of car, truck and wall you physicaly have and a difficulty and you can generates puzzles that fit your game. Additional truck can be purchase with the 'limo' extension raising the truck limit to 5. The two unmovable 1x1 cells called wall/barrier can be easily diy, I made those one usin Fimo dough:

I would recommand you to built those 2 barriers as they unlock quite a lot of puzzle, and introduce a nice diversity to the challenge.

Why the name 'Dino-rush'? : I needed some icon to diferrentiate some decks, and the first things I thought about were dinosaure... No more than that, really :)
